# Third-party code in the shells

The `components/ui/` files in every shell are installed from the shadcn/ui
registry (https://ui.shadcn.com), © shadcn, MIT license. Chart components in
`dashboard/components/charts.tsx` follow the Tremor component patterns
(https://tremor.so), MIT, reimplemented against this repo's tokens on
recharts (MIT). Everything else in `design-data/shells/` is original to this
repository (MIT, see repo LICENSE).
